# Система управления библиотекой

Учебный проект на русском языке: иерархия подразделений, наследование каталога (отдельная таблица на тип), версионирование читателей, доступ к данным через **ОРМ (Prisma)** и **нативный SQL**.

## Требования

- [Node.js](https://nodejs.org/) 20+ (LTS)

Docker и отдельная установка PostgreSQL **не нужны** — база встроена в проект.

## Быстрый запуск

Двойной клик по файлу **`ЗАПУСК.bat`** в корне проекта.

Или в PowerShell:

```powershell
cd "c:\Users\Пользователь\OneDrive\Документы\Библиотека_учебная_практика\backend"
npm install
npx prisma generate
npm run start:fresh
```

Откройте в браузере: **http://localhost:3001**

| Команда | Описание |
|---------|----------|
| `npm run start:fresh` | Полный перезапуск с чистой базой |
| `npm run start:all` | Запуск без сброса данных |

## Структура проекта

| Папка | Содержание |
|-------|------------|
| `database/init.sql` | Схема БД, индексы, тестовые данные |
| `backend/` | Сервер API, Prisma + pg |
| `frontend/` | Веб-интерфейс (русский) |
| `docs/REPORT.md` | Отчёт по учебной практике |

## API (примеры)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/departments/tree/orm` | Иерархия через ОРМ |
| GET | `/api/departments/tree/sql` | Иерархия через нативный SQL |
| GET | `/api/catalog/books/orm` | Книги через ОРМ |
| GET | `/api/catalog/all/sql` | Каталог через SQL (объединение) |
| GET | `/api/readers/:id/versions/sql` | История версий читателя |
| PUT | `/api/readers/:id/orm` | Обновление → новая версия (ОРМ) |
| PUT | `/api/readers/:id/sql` | Обновление → новая версия (SQL) |

## Подключение к БД

```
postgresql://library:library123@127.0.0.1:5433/library_db
```

Порт **5433** — встроенный PostgreSQL (не стандартный 5432).
